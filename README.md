# CODSOFT Task 3 - Responsive Calculator

This project is a responsive web-based calculator built using:

- HTML
- CSS (Grid Layout)
- JavaScript

## Features
- Dark / Light Mode
- Scientific Functions (√, x², %)
- Keyboard Support
- Backspace Functionality
- Calculation History Panel

## Technologies Used
- HTML
- CSS Grid
- JavaScript (DOM Manipulation & Event Handling)

Developed as part of the CODSOFT Internship.
